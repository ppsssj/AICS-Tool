# AI Workspace Test Prompts

기준 데이터:
- 오늘 날짜: 2026-03-08
- 프로젝트
  - `세포 이미징 파이프라인 개편` (`p1`)
  - `로봇 샘플 핸들링 파일럿` (`p2`)
  - `수면 연구 모집 운영` (`p3`)
  - `보관용 분석 벤치마크` (`p4`)

## 빠른 단일 테스트

### 1. 프로젝트 열기

입력:
- `세포 이미징 파이프라인 개편 열어줘`
- `로봇 샘플 핸들링 파일럿으로 이동`
- `보관용 분석 벤치마크 프로젝트 열어줘`

기대 결과:
- 해당 프로젝트 상세 화면으로 이동
- `action.path`가 각각 `/projects/p1`, `/projects/p2`, `/projects/p4`

### 2. 작업 보드 열기

입력:
- `세포 이미징 파이프라인 개편 작업 보드 보여줘`
- `수면 연구 모집 운영 작업 보여줘`
- `로봇 샘플 핸들링 파일럿 task board 열어줘`

기대 결과:
- 각각 `/projects/p1/tasks`, `/projects/p3/tasks`, `/projects/p2/tasks`로 이동

### 3. 문서 화면 열기

입력:
- `세포 이미징 파이프라인 개편 문서 보여줘`
- `로봇 샘플 핸들링 파일럿 docs 열어줘`
- `수면 연구 모집 운영 문서 목록 보여줘`

기대 결과:
- 각각 `/projects/p1/docs`, `/projects/p2/docs`, `/projects/p3/docs`로 이동

참고 문서:
- `이미징 파일명 표준 v2` (`d1`)
- `파일럿 주석 검토 노트` (`d2`)
- `모집 커뮤니케이션 초안` (`d3`)
- `로봇 벤치 사전 점검 체크리스트` (`d4`)

### 4. 일정 화면 열기

입력:
- `세포 이미징 파이프라인 개편 일정 보여줘`
- `수면 연구 모집 운영 이번 주 일정 보여줘`
- `보관용 분석 벤치마크 schedule 열어줘`

기대 결과:
- 각각 `/projects/p1/schedule`, `/projects/p3/schedule`, `/projects/p4/schedule`로 이동

참고 일정:
- `이미징 보정 시간`은 `세포 이미징 파이프라인 개편` 일정
- `모집 스크리닝 콜`은 `수면 연구 모집 운영` 일정

## 현재 프로젝트 컨텍스트 테스트

### 시나리오 A

1. 입력: `세포 이미징 파이프라인 개편 열어줘`
2. 입력: `현재 프로젝트 작업 보드 보여줘`
3. 입력: `현재 프로젝트 문서 보여줘`
4. 입력: `이 프로젝트 일정 열어줘`

기대 결과:
- 순서대로 `/projects/p1` -> `/projects/p1/tasks` -> `/projects/p1/docs` -> `/projects/p1/schedule`

### 시나리오 B

1. 입력: `수면 연구 모집 운영 열어줘`
2. 입력: `이 프로젝트 문서 보여줘`
3. 입력: `현재 프로젝트 일정 확인해줘`

기대 결과:
- 순서대로 `/projects/p3` -> `/projects/p3/docs` -> `/projects/p3/schedule`

아래 질문은 탐색형 테스트에 적합합니다.

입력:
- `문서가 가장 많은 프로젝트 열어줘`
- `보관된 프로젝트 보여줘`
- `이번 주 마감 작업 있는 프로젝트 열어줘`
- `리뷰 대기 작업 있는 프로젝트 보여줘`

기대 포인트:
- `세포 이미징 파이프라인 개편`은 문서 2개, 리뷰 작업 1개
- `보관용 분석 벤치마크`는 `Archived` 상태
- 2026-03-08 기준 마감 임박 작업:
  - `현미경 파일명 규칙 최종 확정` (`p1`, 마감 2026-03-08)
  - `10주차 일정 충돌 조정` (`p3`, 마감 2026-03-09)
  - `온보딩 프로토콜 수정본 검토` (`p1`, 마감 2026-03-10)
